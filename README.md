#### install project dependencies

```js
npm run install-dep
```

#### run project locally

```js
npm run dev
```

#### run tests

```js
 npm test
```

#### requirements

- Node.js v20.6.0 or higher.

#### local

> **default client-localhost http://localhost:3000**

> **default server-localhost http://localhost:4000**
